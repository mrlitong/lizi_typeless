# lizi_typeless 产品方案

> 状态：已定
> 更新日期：2026-07-11（Asia/Shanghai）

## 目标用户与使用场景

目标用户是项目所有者本人，一名经常在浏览器、微信、VS Code 和终端中输入技术内容的开发工程师。

典型场景包括：

1. 口述一段聊天消息或问题描述。
2. 口述开发计划、代码评审意见和技术说明。
3. 混合说出普通话、英文产品名、编程语言、命令、数字和文件路径。
4. 连续口述最长约一分钟的内容，结束后一次性获得整理好的文字。
5. 转录失败后从历史记录找回录音并手动重试，无需重新口述。

## 用户流程

1. 应用随用户需要启动，模型完成一次预热后保持就绪。
2. 用户把光标放在目标输入框中，双击右 Alt。
3. 应用立即开始录音并显示不抢焦点的悬浮提示。
4. 录音期间，悬浮提示展示实时预览；预览是临时结果，不进入目标输入框。
5. 用户再次按下右 Alt。第一次被识别到的结束按键即结束录音，紧随其后的重复按键不触发第二次操作。
6. 应用完成尾部识别、全文定稿和智能整理。
7. 原目标仍安全可用时，应用一次性写入最终文本；否则保存结果并提示用户从历史记录复制。
8. 任何处理失败都保留原始录音，并在历史记录中提供 Retry。

## 交互状态与文案

| 状态 | 用户可见行为 | 建议文案 |
| --- | --- | --- |
| 就绪 | 托盘图标显示服务可用，不展示悬浮窗 | `已就绪` |
| 录音中 | 悬浮窗显示录音状态、时长和滚动预览 | `正在录音` |
| 处理中 | 结束录音，保留悬浮窗并显示当前阶段 | `正在转录` / `正在整理` |
| 写入成功 | 悬浮窗短暂确认后自动收起 | `已输入` |
| 未检测到语音 | 不向目标应用写入文字，记录本次会话 | `未检测到语音` |
| 处理失败 | 明确说明录音已保存，并提供历史入口 | `处理失败，录音已保存` |
| 目标不可用 | 不向其他窗口或 lizi_typeless 自身误写，保留最终文本 | `目标窗口不可写或已变化，请复制结果` |

## 权限与可见性

1. 托盘图标提供当前状态、打开历史记录和退出应用的入口。
2. 悬浮窗始终避免夺取键盘焦点。
3. 应用只处理用户通过右 Alt 主动发起的录音，不持续监听和转录环境声音。
4. 密码框、受保护输入框和高权限窗口不在 MVP 的兼容承诺内。
5. 历史数据只在本机对当前 Windows 用户可见。

## 产品规则与边界场景

### 快捷键

1. 空闲状态下，单击右 Alt 保留系统原有行为；350 ms 内双击右 Alt 开始录音。
2. 录音状态下，第一次按下右 Alt 即结束录音。
3. 结束录音后 650 ms 内的重复按键被视为同一次结束操作，不能重新开始录音。
4. 快捷键识别不能延迟音频采集，也不能漏掉用户开头的第一个字。

### 实时预览与最终写入

1. 实时预览用于确认系统正在工作，不承诺逐字稳定。
2. 临时预览不得修改目标输入框。
3. 只有最终识别和整理成功后才进行一次写入。
4. 开始录音时记录目标窗口；结束时若目标窗口不再安全可用，则停止自动写入，避免内容进入错误应用。
5. lizi_typeless 自身的历史窗口不是自动写入目标；在历史窗口开始的录音只保存结果供复制。

### 智能整理

1. 自动添加标点和合理分段。
2. 删除无意义的口头语、明显重复和已被用户明确推翻的改口内容。
3. 用户明确列举多项内容或语义结构足够清晰时，可以整理成列表。
4. 保留原有观点、语气、事实、数字、专有名词和技术信息。
5. 不回答用户口述的问题，不补充事实，不总结未要求总结的内容。
6. 准确复述与表达美观冲突时，选择准确复述。

### 历史与重试

1. 每次会话保存时间、时长、处理状态、原始录音、原始转录和最终文本。
2. 历史默认永久保留，不自动清理。
3. 失败会话支持播放录音、查看已有文本、Retry、复制和手动删除。
4. 手动 Retry 默认只更新历史结果，不自动写入先前目标窗口，避免错误或重复插入。
5. 已成功写入的会话必须有明确状态，后续重试不得再次自动插入。

## 明确不做的范围

1. 边说边向目标输入框追加或替换临时文本。
2. 语音命令、语音控制电脑和对话式助手。
3. 面向不同应用自动切换数十种写作风格。
4. 商业级新手引导、安装体验和复杂设置中心。
5. 自动上传、同步或删除录音历史。

## 待决项

1. 350 ms 双击间隔和 650 ms 抑制窗口已作为 MVP 基线；只有连续 30 次实机操作显示误触或漏触时才调整。
2. 当前悬浮窗展示模型返回的完整流式假设并限制可见高度；是否改成只显示末尾数行由数日真实使用决定。
